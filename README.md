Sistema básico de gerenciamento de vinhos da Vinharia Agnello, este código simula o cadastro na vinharia exibindo mensagens de boas vindas utilizando alertas, em seguida utiliza prompts para coletar informações essenciais sobre o vinho, como o tipo(Branco, Tinto ou Rosé, a safra(Ano) e a quantidade em estoque. Após a coleta de dados, o programa confirma o cadastro com alertas e exibe todas as informações de forma organizada no console do navegador, utilizando template literals para formatar a saída.
Utilizamos JS para criarmos o sistema de cadastro dos vinhos, e melhoramos a experiência do usuario utilizando CSS.
